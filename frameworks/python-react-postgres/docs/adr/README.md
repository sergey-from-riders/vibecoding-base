# ADR (Architecture Decision Records)

Формат:
- `ADR-0001-title.md`
- `Status: proposed|accepted|deprecated`
- `Context / Decision / Consequences`

В текущем этапе обязательны ADR для:
- изменения стека;
- изменения структуры монорепы;
- изменения auth/session модели;
- изменения модели tenant context.
