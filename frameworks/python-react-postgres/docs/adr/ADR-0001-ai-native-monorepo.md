# ADR-0001: AI-native monorepo baseline

- Status: accepted
- Date: 2026-03-01

## Context
Проект стартует в режиме быстрых итераций с активным использованием coding agents.

## Decision
Принят AI-native baseline:
- contract-first;
- migration-first;
- MCP/agent directories в корне;
- единый API-контракт для web и Qt.

## Consequences
- быстрее onboarding и delivery;
- меньше хаоса в агентных изменениях;
- выше дисциплина документации и тестов.
