# Документация Python React Postgres Framework (2026)

Документация описывает foundation-заготовку под AI-native/vibe coding подход 2026 с текущей границей шаблона:
- только `Auth`;
- только `Switch Company`.

Статусы в этом индексе относятся к готовности заготовки, а не к готовому runtime-приложению.

Корневой [FRAMEWORK.md](../FRAMEWORK.md) описывает общий Kalka delivery framework: как ставить задачи агентам, как проходить lifecycle фичи и какие gate/checkpoint правила обязательны.
