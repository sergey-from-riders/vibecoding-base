# ADR-0007: Next frontend compact high-contrast performance-first style contract

- Status: accepted
- Date: 2026-03-02

## Context
Нужен единый фронтовый стандарт для Next/Node с максимальной производительностью, плотной компоновкой и минимальным визуальным шумом.
Команда требует переиспользование, единый формат списков и строгую дисциплину по стилю/анимациям.

## Decision
Принят жесткий фронтовый стандарт:
- shadcn-only интерактивные примитивы;
- единый list kit;
- максимум `5` цветов на проект;
- `0` shadows, `0` black borders, `0` transition-all;
- mobile-first compact layout;
- анимации без layout shift (кроме drag/reorder сценариев);
- строгие числовые лимиты по размеру файлов/компонентов;
- обязательная автоматическая проверка `tools/scripts/check_web_limits.sh`.

Канонический регламент:
- `docs/19-NEXT-NODE-FRONTEND-STANDARD.md`

## Consequences
- Повышается предсказуемость UI и скорость разработки в многоавторском режиме.
- Снижается количество ad-hoc визуальных решений.
- Требуется строгая дисциплина переиспользования и соблюдения лимитов.
